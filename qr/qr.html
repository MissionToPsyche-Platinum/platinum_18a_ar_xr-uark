NASA Psyche AR — QR + marker sheets (1–4)

  1. Game URL — Paste the full address of your deployed app (the link the QR code should open). Use https:// and include a trailing slash if your host expects it.
  2. Marker images base — Paste the folder URL where the marker PNGs live (must end with 3x3_hamming_6_3/ on your server). Example: https://<user>.github.io/nasa-psyche-ar/assets/3x3_hamming_6_3/
  3. Click Regenerate all (sheets are generated at high pixel resolution), then Print all sheets or Save PNG per card. Scan the QR only, not the marker. A fifth row is QR only (same link, no AR marker) for handouts or labels.

Print at 2 in × 2 in (50.8 mm × 50.8 mm). The AR app expects the printed marker at about that physical size. In your print dialog choose 100% / actual size (not “fit to page” if it shrinks the square). If you use Save PNG, open the file and set width and height to exactly 2 inches (or set a custom paper / crop to a 2″ square). Each browser print page here sizes the square to 2″ × 2″ when you print from this page.

Recommended print: use professional paper (~40 lb) and color printing so blacks, the QR, and any tint in the design stay crisp and consistent for scanning and AR tracking.
